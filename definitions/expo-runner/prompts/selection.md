# Template Selection

Expo React Native mobile app. Native iOS and Android with file-based routing.

Use when:
- You need a native mobile app for iOS and/or Android
- You want React Native with Expo managed workflow
- You want Tailwind-style styling via NativeWind

Avoid when:
- You only need a web app (use Vite templates instead)
- You need a static site or landing page
- You don't need mobile-native features

Built with:
- Expo SDK 52, React Native 0.76, Expo Router (file-based routing)
- NativeWind (Tailwind CSS for React Native), Lucide React Native icons
- TypeScript, Zustand, TanStack Query, Zod, date-fns
